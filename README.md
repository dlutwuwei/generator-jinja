# Description:
Creates a new basic front-end web project for jinja template test and create.

Options:
    Sass: Include Sass

Example:
    yo jinja

    This will create:
        gulpfile.js: Configuration for the task runner.
        bower.json: Front-end packages installed by bower.
        package.json: Development packages installed by npm.

        app/: Your application files.
        test/: Unit tests for your application.
